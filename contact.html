About Us

"In the Moment" was created by Hannah Starcevich, Beth Fulcher, and Andrew Anthony. We are three students currently working our way through UW's Coding Bootcamp. With In the Moment, we were inspired to create a web application that utilized APIs in order to streamline the search experience for finding both relevant events, as well as breweries by those events.


Thanks for using our web application and taking the time to read a little bit about the team behind this project!
